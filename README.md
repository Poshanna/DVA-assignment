# 📊 Instagram OSN Data Storytelling Dashboard

A professional, single-page Streamlit dashboard for analyzing Instagram Online Social Networks (OSN) data. This project focuses on Data Storytelling, engagement patterns, and community network analysis across multilingual content.

## 📱 Project Overview
This dashboard provides a comprehensive analysis of Instagram interaction data, including:
- **Multilingual Content Analysis**: Covering 10 regional languages (English, Hindi, Telugu, Tamil, etc.).
- **Engagement Composition**: Deep dive into Likes, Comments, and Shares.
- **Advanced Visualizations**: PCA Clustering, 3D Engagement Matrix, Radar Comparison Charts, and Trend Analysis.
- **Network Science**: User-Community relationship graphs.
- **Automated Insights**: AI-driven business intelligence for content optimization.

## 🚀 Built With
- **Trae Ecosystem**: AI-native development and rapid prototyping.
- **Streamlit**: Modern, interactive web dashboard framework.
- **Plotly & Seaborn**: High-end interactive visualizations.
- **Scikit-Learn**: Dimensionality reduction (PCA & t-SNE) and data scaling.
- **TextBlob**: Sentiment analysis for multilingual captions.

## 🛠️ Setup and Installation

### Prerequisites
- Python 3.8 or higher
- All required libraries listed in `requirements.txt`

### Installation
1. **Clone the repository**:
   ```bash
   git clone https://github.com/Poshanna/DVA-assignment.git
   cd DVA-assignment
   ```

2. **Install dependencies**:
   ```bash
   python -m pip install -r requirements.txt
   ```

3. **Generate Sample Data**:
   ```bash
   python generate_data.py
   ```

4. **Run the Dashboard**:
   ```bash
   python -m streamlit run app.py
   ```

## 📊 Dashboard Chapters
1. **📈 Visual Story**: Core engagement metrics, performance quadrants, and community influence.
2. **🌐 Regional Pulse**: Language-specific deep dives and influencer discovery.
3. **📊 Comparison**: Radar charts comparing interaction "DNA" across segments.
4. **🧬 Advanced AI**: PCA Clustering and 3D space analysis.
5. **💡 Insights**: Automated findings and data-driven recommendations.
6. **🚀 Ecosystem**: Documentation on the Trae development environment.
